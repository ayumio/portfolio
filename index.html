cARd

AR/VR Featured Web&Mobile Application

cARd is a web and mobile combined application that provides users a new greeting card experience with AR/VR features. We enabled an associated video clips to be streamed on AR/VR view by implementing a redux store into Viro components. We have also implemented a secure media management system with AWS s3 and universal unique ID for senders to upload a video clip.

  • Key Technologies: JavaScript, React Native, Viro React
  • Date: Mar 2019 – Apr 2019

Browse-Keeper

Google Chrome Extension

Browse-Keeper is a Google Chrome Extension that helps users understand and take control of the time they spend on browsing websites. I utilize the Chrome Storage API to manage registered domains and alert users once the time spent exceeds the time limit.

  • Key Technologies: JavaScript, React,Chrome Storage API
  • Date: Mar 2019

Jupiter Tea

E-commerce Website

Jupiter Tea is a well-designed e-commerce site that completed within 7 calendar days with team of four by utilizing NERD stack. It is built with functionalities of listing all items, displaying each individual item, adding products to a cart with desired quantity.
